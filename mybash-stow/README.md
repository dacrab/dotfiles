## ğŸ¨ Material You Integration

The Starship prompt automatically uses Material You colors generated from your current wallpaper:

- **Dynamic theming** - Colors change when you cycle wallpapers
- **Consistent design** - Matches your Hyprland theme
- **Automatic updates** - No manual configuration needed
- **Generated by matugen** - Uses the same color extraction as Hyprland

### How it works:
1. When you change wallpapers using `wallpaper cycle`, matugen generates colors
2. The `generate_starship_colors` script extracts colors from matugen output
3. Starship configuration is automatically updated with new colors
4. Your prompt instantly reflects the new theme
